# 🌟 이준캐치

몬스터를 잡고 레벨을 올리는 캐주얼 캐치 게임.
포켓몬 스타일의 캐릭터가 볼을 던져 다양한 희귀도의 몬스터를 포획합니다.
5마리 잡을 때마다 덧셈 퀴즈가 등장하는 교육적 요소도 포함되어 있습니다.

**🔗 배포 주소:** [wildcatch.vercel.app](https://wildcatch.vercel.app)

---

## 게임 방법

| 조작 | 키보드 | 모바일 |
|------|--------|--------|
| 이동 | ← → 방향키 | ◀ ▶ 버튼 |
| 볼 던지기 | Space | 던지기! 버튼 |

1. 좌우로 이동해 몬스터 방향을 조준
2. Space(또는 던지기 버튼)로 볼 발사
3. 볼이 몬스터에 맞으면 포획 시도
4. 성공하면 XP 획득 → 볼 레벨업
5. **5마리마다 덧셈 퀴즈** 등장 — 정답을 맞춰야 게임 계속

## 주요 기능

### 몬스터 시스템
**100종**의 몬스터, 희귀도 5단계 (레벨별 10종씩 랜덤 등장):

| 희귀도 | 레벨 | 예시 |
|--------|------|------|
| Common   | 1–3  | 꼬물이 🐛 꿀벌이 🐝 여우린 🦊 펭귄이 🐧 외 27종 |
| Uncommon | 4–5  | 토실이 🐰 코알라 🐨 캥거루 🦘 하마야 🦛 외 10종 |
| Rare     | 6–7  | 호랑이 🐯 사자왕 🦁 유니콘 🦄 고래신 🐋 외 10종 |
| Epic     | 8–9  | 드래곤 🐲 번개신 ⚡ 태양신 ☀️ 은하수 🌌 외 10종 |
| Legend   | 10   | 별이 🌟 무지개신 🌈 왕의신 👑 빛의신 ✨ 외 6종 |

**⏰ 몬스터 제한 시간**: 15초 안에 포획해야 함. 5초 이하 남으면 화면 중앙에 카운트다운 표시. 시간 초과 시 몬스터 도망 (miss 미적용) + ⚠️ 위기 카운트다운 10초 발동 — 다음 몬스터를 10초 안에 잡지 못하면 게임 오버.

**👾 몬스터 행동 패턴** (레벨별 랜덤 배정):

| 패턴 | 조건 | 특징 |
|------|------|------|
| 졸음형 | Lv 1–3 (35% 확률) | 주기적으로 멈추고 Zzz 표시 — 잠든 동안 포획률 2배! |
| 점프형 | Lv 3–6 (40% 확률) | 90프레임마다 위로 크게 점프 |
| 지그재그 | Lv 7+ (40% 확률) | 25프레임마다 방향 전환 |
| 도망 AI | Lv 5+ (항상) | 볼이 날아오면 반대 방향으로 이동 |

**💀 보스 몬스터**: 50마리 포획마다 대왕 보스 👑 등장. 크기 2배, 속도 1.5배, HP 2 (2번 맞춰야 포획). 황금볼 또는 뽑기권 활용 추천.

### 볼 레벨 시스템
XP를 쌓아 볼을 업그레이드. 볼 레벨이 높을수록 포획률 상승.

`WildBall → SuperBall → HyperBall → MegaBall → UltraBall → MasterBall → DarkBall → StarBall → CosmoBall → OmegaBall`

### 캐릭터 레벨 시스템
**5마리** 포획마다 캐릭터 레벨 1 상승 (최대 Lv.50). Lv.10 / 20 / 30 / 40 구간 돌파는 10마리 필요.
레벨 구간별 복장 색상과 모자 형태가 바뀌며 파티클 이펙트와 오라 효과 발동.

| 레벨 | 복장 | 모자 |
|------|------|------|
| Lv. 1–10 | 블루 | 기본 캡 🧢 |
| Lv. 11–20 | 레드 | 탐험가 모자 (넓은 챙 + 컬러 밴드) |
| Lv. 21–30 | 퍼플 | 마법사 고깔모자 🧙 |
| Lv. 31–40 | 골드 | 3포인트 왕관 👑 |
| Lv. 41–49 | 다크 | 에픽 5포인트 황금 왕관 |
| Lv. 50 | 특별 (주황) | 에픽 왕관 + 별★ |

### 몬스터 속도 난이도 시스템
캐릭터 레벨 5단계마다 몬스터 이동 속도가 증가.

| 레벨 구간 | 속도 배율 |
|----------|---------|
| Lv. 1–5 | 1.0× (기본) |
| Lv. 6–10 | 1.12× |
| Lv. 11–15 | 1.24× |
| Lv. 16–20 | 1.36× |
| Lv. 26–30 | 1.60× |
| Lv. 41+ | 최대 2.0× |

### 낮/밤 배경 사이클
2분 주기로 낮 ↔ 밤이 자동 전환.
하늘 색상, 태양/달/별, 잔디 색이 부드럽게 변화.

### 🎁 아이템 시스템 (6종)
포획 성공 후 35% 확률로 아이템 등장. 볼로 맞추면 즉시 효과 발동. 5초 안에 못 맞추면 도망.

| 아이템 | 효과 |
|--------|------|
| ⚡ 빠르게 | 5초간 플레이어 이동속도 2배 |
| 🐌 느리게 | 5초간 몬스터 속도 35%로 감소 |
| 🧲 자석   | 5초간 몬스터가 플레이어 방향으로 이동 |
| 🛡️ 방패   | 다음 볼 miss 1회 무효 |
| ⏰ 시간+  | 몬스터 제한 시간 +10초 |
| 🎫 뽑기권 | 현재 몬스터 즉시 자동 포획 |

### 🌈 골든 타임
20마리 포획마다 30초간 골든 타임 발동. XP 2배 획득, 화면에 황금빛 오버레이 + 상단 배너 표시.

### 🎆 이펙트
- **포획 성공**: 화면 플래시 + 몬스터별 전용 파티클 폭발
- **콤보 팝업**: 3콤보 이상 시 화면 중앙에 팡! 애니메이션
- **황금볼**: 발사 시 무지개 궤적(rainbow trail)
- **몬스터 도망**: 😂 웃음 이모지 파티클

### 🧮 교육 퀴즈
5마리 포획마다 덧셈/뺄셈 퀴즈 등장. 1턴 덧셈(+) → 1턴 뺄셈(-) 교대.
4지선다 버튼으로 선택. 정답을 맞춰야 게임 재개.
