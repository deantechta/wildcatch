# 🌟 이준캐치

몬스터를 잡고 레벨을 올리는 캐주얼 캐치 게임.
포켓몬 스타일의 캐릭터가 볼을 던져 다양한 희귀도의 몬스터를 포획합니다.
5마리 잡을 때마다 덧셈 퀴즈가 등장하는 교육적 요소도 포함되어 있습니다.

**🔗 배포 주소:** [wildcatch.vercel.app](https://wildcatch.vercel.app)

---

## 게임 방법

| 조작 | 키보드 | 모바일 |
|------|--------|--------|
| 이동 | ← → 방향키 | ◀ ▶ 버튼 |
| 볼 던지기 | Space | 던지기! 버튼 |

1. 좌우로 이동해 몬스터 방향을 조준
2. Space(또는 던지기 버튼)로 볼 발사
3. 볼이 몬스터에 맞으면 포획 시도
4. 성공하면 XP 획득 → 볼 레벨업
5. **5마리마다 덧셈 퀴즈** 등장 — 정답을 맞춰야 게임 계속

## 주요 기능

### 몬스터 시스템
**100종**의 몬스터, 희귀도 5단계 (레벨별 10종씩 랜덤 등장):

| 희귀도 | 레벨 | 예시 |
|--------|------|------|
| Common   | 1–3  | 꼬물이 🐛 꿀벌이 🐝 여우린 🦊 펭귄이 🐧 외 27종 |
| Uncommon | 4–5  | 토실이 🐰 코알라 🐨 캥거루 🦘 하마야 🦛 외 10종 |
| Rare     | 6–7  | 호랑이 🐯 사자왕 🦁 유니콘 🦄 고래신 🐋 외 10종 |
| Epic     | 8–9  | 드래곤 🐲 번개신 ⚡ 태양신 ☀️ 은하수 🌌 외 10종 |
| Legend   | 10   | 별이 🌟 무지개신 🌈 왕의신 👑 빛의신 ✨ 외 6종 |

**⏰ 몬스터 제한 시간**: 15초 안에 포획해야 함. 5초 이하 남으면 화면 중앙에 카운트다운 표시. 시간 초과 시 몬스터 도망 (miss 미적용) + ⚠️ 위기 카운트다운 10초 발동 — 다음 몬스터를 10초 안에 잡지 못하면 게임 오버.

**👾 몬스터 행동 패턴** (레벨별 랜덤 배정):

| 패턴 | 조건 | 특징 |
|------|------|------|
| 졸음형 | Lv 1–3 (35% 확률) | 주기적으로 멈추고 Zzz 표시 — 잠든 동안 포획률 2배! |
| 점프형 | Lv 3–6 (40% 확률) | 90프레임마다 위로 크게 점프 |
| 지그재그 | Lv 7+ (40% 확률) | 25프레임마다 방향 전환 |
| 도망 AI | Lv 5+ (항상) | 볼이 날아오면 반대 방향으로 이동 |

**💀 보스 몬스터**: **7마리 포획마다** 픽셀아트 보스 등장. 30종 포켓몬 중 랜덤 선택. 크기 3배, HP는 캐릭터 레벨에 따라 증가. 포획 성공 시 👑 배너 등장.

| 보스 등장 주기 | 7마리마다 |
|-------------|---------|
| 보스 레벨 | 캐릭터 Lv. 1–9: 10 / Lv. 10–19: 11 / … / Lv. 50: 15 |
| 일반 보스 HP | 7 + (캐릭터 Lv ÷ 10) |
| 파워 보스 HP | 12 + (캐릭터 Lv ÷ 10) |
| 공격 방식 | 속성별 투사체 발사 (불/물/전기/고스트/풀) |

**보스 포켓몬 개성 시스템**: 포켓몬마다 기동력·공격빈도·회피율이 다름.

| 컨셉 | 예시 |
|------|------|
| 고속 맹공격 | 피카추 (기동력↑, 자주 공격) |
| 느리지만 탱커 | 거북왕★ (느리지만 HP 높고 회피↑) |
| 극도 회피형 | 팬텀 (매우 빠르고 최종 포획 회피율 15%) |
| 느리지만 공세 | 파이리 (공격 쿨타임 가장 짧음) |

**보스 전투 특징**:
- 피격할수록 속도가 점차 감소 (고레벨일수록 최저 속도 하한이 높아 끝까지 긴장감 유지)
- 같은 자리에서 연속 피격 시 순간이동 (속도 완전 리셋)
- HP ≥ 2: 최소 50% 확률로 데미지 적용 (볼 레벨 격차 무관)
- HP = 1: 기존 포획률 적용 + 일부 포켓몬 회피 가능 (황금볼로 회피 무시 가능)

### 볼 레벨 시스템 (20단계)
XP를 쌓아 볼을 업그레이드. 볼 레벨이 높을수록 포획률 상승 + **발사 이펙트 강화**.

`WildBall → SuperBall → ... → OmegaBall → PrimeBall → AbyssBall → NovaBall → PlasmaBall → FrostBall → NebulaBall → ShadowBall → VoidBall → CometBall → GodBall`

| 볼 레벨 | 이펙트 |
|---------|--------|
| Lv 1–2 | 기본 볼 (소프트 글로우) |
| Lv 3–4 | 컬러 trail 잔상 |
| Lv 5–7 | 밝은 trail + 스파크 3~5개 |
| Lv 8–10 | 무지개 rainbow trail + 레인보우 스파크 |
| Lv 11–12 | 비비드 레드/딥 마젠타 shimmer trail |
| Lv 13–14 | 노바 스타버스트 / 네온 그린 플라즈마 |
| Lv 15–16 | 아이스 크리스탈 / 플라즈마 + 회전 글로우 링 |
| Lv 17–18 | 민트 씨포 / 딥 스페이스 퍼플 + 회전 링 |
| Lv 19 | 혜성 — 황금+불꽃 합체 궤적 |
| Lv 20 (GodBall) | 풀 레인보우 다층 이펙트 + 외부 펄스 링 |
| 포획 중 orbit | 볼 레벨에 따라 공 수, 속도, 색상 변화 |

### 캐릭터 레벨 시스템 (XP 방식)
몬스터 포획 시 **몬스터 레벨**만큼 XP 획득. XP 누적으로 캐릭터 레벨업 (최대 Lv.50).

| 레벨 구간 | 레벨당 필요 XP |
|----------|-------------|
| Lv. 1 → 10 | 10 XP (lv1 몬스터 약 10마리) |
| Lv. 10 → 20 | 100 XP (lv5 몬스터 약 20마리) |
| Lv. 20 → 30 | 300 XP (lv7 몬스터 약 43마리) |
| Lv. 30 → 40 | 600 XP (lv8 몬스터 약 75마리) |
| Lv. 40 → 50 | 900 XP (lv9 몬스터 약 100마리) |

레벨 구간별 복장 색상과 모자 형태가 바뀌며 파티클 이펙트와 오라 효과 발동.

### 볼 발사 속도 자동 성장
볼 레벨이 오를수록 발사 속도 증가 (2단계 선형 증가, 일반 최대 350% 캡).
캐릭터 레벨 10단계마다 추가 +10% 보너스 (Lv10: +10%, Lv50: +50%).
⚡빠르게 아이템 사용 시 예외적으로 최대 380% 도달 가능!

| 볼 레벨 | 속도 |
|---------|------|
| Lv 1 | 100% |
| Lv 5 | 136% |
| Lv 10 | 181% |
| Lv 15 | 265% |
| Lv 20 | 350% (일반 Max) |
| 빠르게 아이템 | 최대 380% |

| 레벨 | 복장 | 모자 |
|------|------|------|
| Lv. 1–10 | 블루 | 기본 캡 🧢 |
| Lv. 11–20 | 레드 | 탐험가 모자 (넓은 챙 + 컬러 밴드) |
| Lv. 21–30 | 퍼플 | 마법사 고깔모자 🧙 |
| Lv. 31–40 | 골드 | 3포인트 왕관 👑 |
| Lv. 41–49 | 다크 | 에픽 5포인트 황금 왕관 |
| Lv. 50 | 특별 (주황) | 에픽 왕관 + 별★ |

### 몬스터 속도 난이도 시스템
캐릭터 레벨 5단계마다 몬스터 이동 속도가 증가.

| 레벨 구간 | 속도 배율 |
|----------|---------|
| Lv. 1–5 | 1.0× (기본) |
| Lv. 6–10 | 1.12× |
| Lv. 11–15 | 1.24× |
| Lv. 16–20 | 1.36× |
| Lv. 26–30 | 1.60× |
| Lv. 41+ | 최대 2.0× |

### 낮/밤 배경 사이클
2분 주기로 낮 ↔ 밤이 자동 전환.
하늘 색상, 태양/달/별, 잔디 색이 부드럽게 변화.

### 🎁 아이템 시스템 (14종)
포획 성공 후 35% 확률로 아이템 등장. 볼로 맞추면 즉시 효과 발동. 5초 안에 못 맞추면 도망. 전 아이템 균등 확률.

| 아이템 | 효과 |
|--------|------|
| ⚡ 빠르게 | 5초간 볼 발사 속도 +30% (최대 380% 예외 허용) |
| 🐌 느리게 | 5초간 몬스터 속도 35%로 감소 |
| 🧲 자석   | 5초간 몬스터가 플레이어 방향으로 이동 |
| 🔵 볼확대  | 5초간 볼 히트박스 2배 (맞추기 쉬워짐!) |
| ⏰ 시간+  | 몬스터 제한 시간 +10초 |
| 🎫 뽑기권 | 나머지 13종 중 랜덤 1개 발동 |
| ⏪ 되감기 | 볼이 화면 밖으로 나가도 플레이어 위치로 되돌아옴 |
| 🎯 조준경 | 5초간 볼이 몬스터 방향 자동 추적 |
| 🔥 콤보불꽃 | 10초간 포획마다 콤보 +2씩 적립 |
| 🧊 냉동   | 3초간 몬스터 완전 정지 |
| 💫 더블   | 다음 포획 1회 XP·점수 ×3 |
| 🏃 무빙패스트 | 5초간 이동속도 50% 증가 |
| 💊 회복   | HP 2 즉시 회복 (최대 초과 불가) |
| 🔱 멀티샷  | 5초간 볼 3개 동시 발사! (좌·중앙·우) |
| 💝 도움   | 7번 연속 miss 시 자동 등장 |

### 🌈 골든 타임
20마리 포획마다 30초간 골든 타임 발동. XP 2배 획득, 화면에 황금빛 오버레이 + 상단 배너 표시.

### 🎆 이펙트
- **포획 성공**: 화면 플래시 + 몬스터별 전용 파티클 폭발
- **콤보 팝업**: 3콤보 이상 시 화면 중앙에 팡! 애니메이션
- **황금볼**: 발사 시 무지개 궤적(rainbow trail)
- **몬스터 도망**: 😂 웃음 이모지 파티클

### 🧮 교육 퀴즈
5마리 포획마다 덧셈/뺄셈 퀴즈 등장. 1턴 덧셈(+) → 1턴 뺄셈(-) 교대.
4지선다 버튼으로 선택. 정답을 맞춰야 게임 재개.

- 공격 쿨타임에 하한 클램핑 적용 (하드 최소 1.5초 / 이지 최소 3초)
